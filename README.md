#  Calculadora de BF (Marinha Americana) em Java

Este projeto é uma ferramenta de linha de comando desenvolvida em Java para estimar o percentual de gordura corporal (Body Fat) utilizando o protocolo da Marinha dos EUA. 

O método é uma alternativa prática ao uso de adipômetros, utilizando apenas medidas de circunferência corporal para calcular a composição física.

------------------------------------------------------------------------

## 📌  Funcionalidades

* **Cálculo por Gênero:** Fórmulas matemáticas distintas para homens e mulheres.
* **Protocolo da Marinha:** Utiliza a diferença entre cintura e pescoço (e quadril para mulheres) em relação à altura.
* **Interface Simples:** Interação direta via console (Scanner).
* **Precisão Numérica:** Exibição do resultado final formatado com duas casas decimais.

------------------------------------------------------------------------

## 🛠️ Como Executar o Projeto

1.  **Clone o repositório:**
    ```bash
    git clone https://github.com/GustavoLopesMoreira/Calculadora-de-BF.git
    ```

2.  **Entre na pasta do código fonte:**
    ```bash
    cd Calculadora-de-BF/src
    ```

3.  **Compile o arquivo Java:**
    ```bash
    javac calculadora_de_bf.java
    ```

4.  **Execute o programa:**
    ```bash
    java calculadora_de_bf
    ```
